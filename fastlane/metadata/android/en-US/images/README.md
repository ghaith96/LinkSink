# F-Droid Icon Requirements

## icon.png

Create a 512x512 PNG icon and place it here as `icon.png`.

### Option 1: Export from Android Studio
1. Open the project in Android Studio
2. Right-click `res` folder → New → Image Asset
3. Configure your icon design
4. Export at 512x512 resolution

### Option 2: Use Android Asset Studio
1. Visit https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html
2. Design or upload your icon
3. Download and extract the 512x512 version

### Option 3: Create custom icon
Design a 512x512 PNG icon using any graphics tool (Figma, Inkscape, etc.)

## phoneScreenshots/
